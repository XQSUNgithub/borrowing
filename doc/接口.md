
用户表


账号<br/>
uuid verchar(36)<br/>
password verchar(16)<br/>
限制6-16字符<br/>
state bit (0 管理者 1 普通)<br/>
realname verchar(10)<br/>

login/register
post
String realname<br/>
String password<br/>
以上两个字段base64加密后作为ajax的data传给后端<br/>
直接返回token
放在结果类的data字段里

login/do_login
post
String realname<br/>
String password<br/>
以上两个字段base64加密后作为ajax的data传给后端<br/>
直接返回token
放在结果类的data字段里

用户管理界面<br/>

/query<br/>
请求方式：post<br/>
action:"1",
tableName:"User",
key:"uuid",
keyword:""
keyword为空时视为list
返回list<>
